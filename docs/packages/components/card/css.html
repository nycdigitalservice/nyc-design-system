Primary action

The primary action is associated with an a or button element and makes the entirety of the card clickable.

Inspect code

.card__primary-action,
.card__primary-action:link {
  display: flex;
  align-items: flex-start;
  color: currentColor;
  text-decoration: none;
  &:before {
    content: '';
    position: absolute;
    top: 0; /* 2 */
    left: 0; /* 2 */
    right: 0; /* 2 */
    bottom: 0; /* 2 */
    Z-index: 1; /* 3 */
  }
  & > :first-child {
    flex: 1;
    display: -webkit-box;
    -webkit-line-clamp: 3; /* number of lines to show */
    line-clamp: 3;

    text-decoration: underline;

Container query

To adjust the cards presentation based on its width, we use a Container Query

Inspect code

/* @container grid-container (max-width: 768px) { */
/*   .card { */
/*     flex-direction: row; */
/*     flex-wrap: wrap; */
/*     overflow: hidden; */
/*   } */
/*   .card__header { */
/*     flex: 1; */
/*     flex-basis: 100%; */
/*   } */
/*   .card__media { */
/*     width: 35%; */
/*     height: 100%; */
/*     position: absolute; */
/*     & img { */
/*       width: 100%; */
/*       height: 100%; */
/*       object-fit: cover; */
/*     } */
/*   } */
/*   .card__body { */
/*     flex-grow: 1; */
/*     flex-basis: 100%; */
/*   } */
/*   .card:has(.card__media) { */
/*     & .card__header, */
/*     & .card__body { */
/*       margin-left: auto; */
/*       flex-basis: 65%; */
/*       flex-grow: 0; */
/*     } */
/*   } */
/* } */
@container card-container (min-width: 320px) {
  .card__media {
    width: 35%;
    height: 100%;
    position: absolute;
    aspect-ratio: 1/1;
    & img {
      width: 100%;
      height: 100%;
      object-fit: cover;
    }
  }
  .card__body {
    flex-grow: 1;
    flex-basis: 100%;
  }
  .card:has(.card__media) {
    & .card__header,
    & .card__body {
      margin-left: auto;
      flex-basis: 65%;
      flex-grow: 0;
      width: 65%;
    }
  }    
}
